MINECRAFT

Minecraft is a 2011 sandbox video game created by Swedish game developer Markus Persson and later developed by Mojang. The game allows players to build with a variety of different blocks in a 3D procedurally generated world, requiring creativity from players. It also has features like Destroying,PVP/PVE,crafting,exploring and much more!

What was the first name of Minecraft?
Pre-classic is the name given to the prototype versions developed before Classic. It was the very first development phase in Minecraft, lasting for a week. The phase was originally named "Cave Game" until the game was changed to "Minecraft: Order of the Stone" then to "Minecraft".

What is Minecraft coded in?
Mods written for Minecraft are also written in Java. Some actually use Scala, an alternative language for the JVM, as well. The versions of Minecraft that run on game consoles, mobile devices, and the Raspberry Pi, though, are all written in native code, likely C++.
